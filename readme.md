Laravel与表达是一个web应用程序框架,优雅的语法。 我们相信发展必须是愉快的,创造性的体验真正的满足。 Laravel试图把痛苦从发展的宽松常见任务中使用的大多数web项目,如身份验证、路由、会话、排队和缓存。

Laravel是可访问的,但功能强大,提供所需的工具大、健壮的应用程序。 一个极好的控制反转容器,富有表现力的移民系统,紧密集成单元测试支持给你的工具,你需要构建任何应用程序任务。
